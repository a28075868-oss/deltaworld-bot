📝 ИНСТРУКЦИЯ:

1. Зайдите в Railway Dashboard
2. Откройте проект Python бота
3. Settings → Variables
4. Для каждой переменной:
   • Нажмите "New Variable"
   • Введите имя (например: DB_HOST)
   • Введите значение (например: reseau.proxy.rlwy.net)
   • Нажмите "Add"

5. После добавления всех переменных Railway автоматически
   перезапустит бот с новыми настройками!
